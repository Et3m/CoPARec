# Corrective Aggregation and Preference-Aware for Multimodal Recommendation Systems

## Introduction

This is the Pytorch implementation for our paper: Corrective Aggregation and Preference-Aware for Multimodal Recommendation Systems

### Enviroment Requirement

Python==3.10,

Pytorch==2.4.0.

## 1. Download data

Downloading from this link [Google Drive](https://drive.google.com/drive/folders/1BxObpWApHbGx9jCQGc8z52cV3t9_NE0f?usp=sharing).

## 2. Training on a local server using PyCharm.

Run CoPARec by ``python main.py`` with the default dataset as Baby. Specific dataset selection can be modified in `main.py`.

## 3. Training on a local server using Git Bash.

Run CoPARec by ``train.\`` with the default dataset is Baby. Specific dataset selection can be modified in `train.py`.
